# Paths

The paths module consists of:

- Linear
- Circular
  - Spiral
- Bezier

And it's corresponding "followers". (Scripts to move the object around the trajectory)

The bezier curve editor is an implementation by [Sebastian Lague](https://github.com/SebLague/)
